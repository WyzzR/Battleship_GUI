# Battleship GUI

Battleship game with GUI using pygame

## Files

- Main menu: `main.py`
- Play: `play.py`
- Bot brains: `bot.py`

## Requirements

- Pygame
- libmpg123-0.dll (in `C:\Windows\System32` and `C:\Windows\SysWOW64` to allow pygame to play mp3 files)
